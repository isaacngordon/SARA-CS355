Welcome!

This site hold the contents of a project made by Isaac Gordon and Ron Refael, which was built for Professor Lawrence Teitleman's course in Web and Internet Technologies.

What is SARA 355?

SARA stands for "Search and Reoprt Application", and within this site are several pages demonstrating different kinds of SARAs.

What Technologies Do We Use?

There are many different technologies used in the evelopment of websites and web applications. The following are the technologies utilized in this site thus far in the development of this site:

  • HTML5
  • CSS3
  • Javascript

Availible Searchs

Dummy Search

Check out a static search page, which demostrates the layout of a search results page styled in this website's CSS stylesheet.

File Search

As the internet is a colleciton of hyperlinked files across amny machines, it is important for use to learn how to search through the content of files and documents if we are going to build a search engine. On this portion of SARA 355, we will setup a search function that will return results from CSV, JSON, and XML files.

Google API Search

Search the internet utilizing the Google Search API. Google offers a huge suite of application programming interfaces that can be used to accomplish many, many, webtasks without having to write your own functionality.
As we learn how search engines function, we will use the Google Search API to offload some of the hard work.

The Real Deal Search

This is the real deal SARA; the one we've been building up to. This search page was built to comb websites for certain identifying information. It was built from scratch, using principles learned through the above iterations.
